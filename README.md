A statistical study analyzing employees’ online shopping preferences using Python. Includes Chi-Square, Z-Test, and t-Test to compare behavior, spending, payment methods, and platform choices, supported by visualizations and survey-based insights.

Overview:
This project explores how employees shop online, identifying patterns such as:
Preferred shopping platforms
Reasons for choosing online shopping
Product categories purchased
Review-reading habits
Monthly spending differences
Payment method preferences
Fraud/security experiences

Statistical Methods:
Chi-Square Test
Two-Proportion Z-Test
Two-Sample t-Test
Graphical analysis (bar charts, pie charts, boxplots)

Technologies Used:
Python
Pandas
NumPy
Matplotlib
Statsmodels
SciPy
OpenPyXL

How to Run:
Install dependencies
Add dataset 
Run analysis scripts
